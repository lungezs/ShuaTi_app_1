## 1. 项目名称
**从0开始制作一个刷题 App**（可运行安卓 App）

---

## 2. 核心功能
一个**安卓刷题软件**，支持：
- 📥 导入 Excel 文件（.xlsx / .xls）作为题库
- 📝 三种题型：单选题、多选题、判断题
- 📖 顺序练习、收藏练习、错题本、模拟考试
- 🎯 答题颜色反馈（绿色正确 / 红色错误）
- ⭐ 题目收藏（独立于练习进度）
- 📕 错题自动收录（顺序/收藏/模拟考试中答错的题）
- 📊 模拟考试（自定义题型数量，提交后显示得分与解析）
- 💾 本地存储（做题记录、收藏、错题本全部持久化）
- 🚀 已打包为 `.apk` 可安装文件

---

## 3. 技术环境
- **开发工具**：
  - 电脑：VS Code + GitHub Codespaces
  - 手机：Termux + MT管理器
  - 调试：Expo Go
- **编程语言**：JavaScript（React Native）
- **打包工具**：EAS Build（Expo Application Services）

## 4. 项目文件结构
```
ShuaTi_app_1/
└── myapp_1/                          # 主项目文件夹
    ├── App.js                         # 主路由（首页/模式选择/考试路由）
    ├── app.json                       # Expo 配置文件（含应用名、图标、包名）
    ├── package.json                   # 依赖管理
    ├── eas.json                       # EAS 打包配置
    ├── assets/                        # 静态资源
    │   └── icon.png                   # 应用图标（1024x1024）
    ├── screens/                       # 页面组件
    │   ├── QuizScreen.js              # 刷题界面（顺序/收藏/错题本）
    │   ├── QuizModeScreen.js          # 模式选择界面
    │   ├── ExamSetupScreen.js         # 模拟考试设置
    │   └── ExamScreen.js              # 模拟考试答题与结果
    ├── hooks/                         # 自定义 Hooks
    │   └── useQuestionBank.js         # 题库状态管理 + 收藏/错题本逻辑
    ├── utils/                         # 工具函数
    │   └── excelHelper.js             # Excel 解析（支持 xlsx / xls）
    └── styles/                        # 全局样式
        └── commonStyles.js            # 所有页面共用样式
```
